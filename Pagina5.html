INSTALACIÓN DEL FTP

Instalacion FTP

Para instalar FTP utilizaremos los sigueintes comandos

apt-get update
apt-get install proftpd

Configuracion FTP

Lo primero que debemos hacer para que el FTP sea funcional es crear un usuario
por suerte FTP utiliza los usuarios asique nosotros utilizaremos nuestro usuario
grupo3 pero podeis crear un nuevo usuario con el sigueinte comando

adduser ftpdaw -shell /bin/false -home /var/www/daw.com/public_html

lo siguiente sera configurar el archivo proftpd.conf

nano /etc/proftpd/proftpd.conf

añadimos las sigueintes lineas en el apartado global
RootLogin off
RequireValidShell off

reiniciamos el servicio

service proftpd restart

por ultimo añadimos las sigueintes linas al usuario que queramos dentro del archvio de configuracion
en nuestro utilizaremos el usuario anonimo

<Anonymous /var/www/daw.com/public_html/anonimo>
User ftp
Group nogroup
<Limit LOGIN>
AllowAll
</Limit>
UserAlias anonymous ftp
DirFakeUser on ftp
DirFakeGroup on ftp
RequireValidShell off
MaxClients 10
<Directory *>
<Limit WRITE>
DenyAll
</Limit>
</Directory>
</Anonymous>
