INSTALACIÓN DEL DNS

Instalacion DNS

Para instalar DNS utilizaremos el sigueinte comando

apt-get install dnsmasq

configuracion DNS

Primero debemos entrear al archivo de configuracion de DNS

nano /etc/resolv.dnsmasq.conf
Añadimos la siguiente linea para configurar la DNS de Cache

nameserver 10.9.55.3
Una vez hecho vamos al archivo dnsmasq.conf
y cambiamos la linea

#resolv-file=
Por

resolv-file=/etc/resolv.dnsmasq.conf

lo sigueinte sera añadir nuestro servidor al archivo resolv.conf

nano /etc/resolv.conf

lo dejamos con esta sola linea

Por ultimo añadimos los parametros que queremos al archivo hosts
10.9.52.108:80 www.wiki2.com

en nuestro caso se quedaria asi
Por ultimo cada vez que añadamos un host deberemos reiniciar el dns

service dnsmasq restart
