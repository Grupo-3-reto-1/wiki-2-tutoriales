APACHE2

Apache instalacion

Para instalar el apache en nuestro Debian debemos utilizar los siguientes comandos

apt-get update
apt-get install apache2 apache2-doc apache2-utils

Apache Configuracion

Una vez instalado el apache el proximo paso sera configurarlo para ello ejecutamos el siguiente comando

nano /etc/apache2/apache2.conf

Esto nos llevara al archivo de configuracion de apache donde deberemos especificar nuestra configuracion:
en nuestro caso seria la siguiente dentro del Ifmodule

StartServers 5
MinSpareServers 5
MaxSpareServers 10
MaxClients 150
MaxRequestsPerChild 5000

Apache Configuracion virtual host

habiendo configurado ese aspecto del Apache deberemos crear un Virtual Host para nuestra pagina
configurarlo

sudo nano /etc/apache2/sites-available/Wiki2.com.conf

En este archivo incuimos la siguiente configuracion dentro del virtual host con nuestra IP

DocumentRoot /var/ww/html/retowiki2/reto-1-
ServerName grupo3wiki2.com
DirectoryIndex Pagina1.html
