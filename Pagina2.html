DEBIAN

Debian Instalacion y SetUP

Debian es distinto a Ubuntu en algunos aspectos pero la instalacion incial es muy parecida.
A la hora de instalar lo mas importante es tener sufuciente ram en nuestra maquina para no
tener que hacerlo en modo comando


Una vez instalado Debian deberemos hacer unas configuraciones iniciales.

Tendremos configurar las interfaces de red;
Para ello tenemos que ejecutar el siguiente comando:
sudo nano /etc/networks/interfaces
Esto abrira el archivo de configuracion de las interfaces y lo editaremos de la siguiente manera :

auto ens18
iface ens18 inet static
address 10.9.52.108 (esta es la ip que vamos a tener)
netmask 255.255.255.0
network 10.9.52.0
broadcast 10.9.52.255
gateway 10.9.52.108
